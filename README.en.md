# HiverX

#### Description
🎉🔥HiverX是基于Spring Boot 2.x的一站式前后端分离快速开发平台。支持工作流、License授权、设备管控，可视化大屏，报表设计器，表单设计器，代码生成器等功能。能帮助你快速建立IOT物联网平台、MES生产管理平台等相关业务系统。
